# uniapp-achieve

> [DCloud `uni-app` 微信小程序开发框架。](https://uniapp.dcloud.io/quickstart)

## 搭建项目框架

### 初始化项目

```node
vue create -p dcloudio/uni-preset-vue uniapp-achieve
```

### 运行微信小程序项目

```node
npm run dev:mp-weixin
```

### 编译打包项目

```node
npm run build:mp-weixin
```
